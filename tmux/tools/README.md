# Tmux Tools

Herramientas auxiliares para tmux.

## Incluido

- `prime/tmux-sessionizer.sh`: selector generico de carpetas con `fzf` que crea o cambia a una sesion tmux.

## Pendiente

- Integrar el sessionizer con Ghostty o Kitty.
- Decidir raices de busqueda definitivas.
- Crear un selector SSH propio si realmente hace falta.

## Uso

```bash
~/mydotfiles/tmux/tools/prime/tmux-sessionizer.sh
~/mydotfiles/tmux/tools/prime/tmux-sessionizer.sh ~/mydotfiles
~/mydotfiles/tmux/tools/prime/tmux-sessionizer.sh search ~/Developer
```
