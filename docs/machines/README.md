# Machine Notes

Documentacion operativa de maquinas concretas: inventario, estado de migracion,
riesgos, recuperacion y tareas pendientes.

Esta carpeta no contiene configuraciones desplegables. Las piezas reutilizables
pertenecen a `shared/` u `os/`; la seleccion para cada entorno pertenece a
`profiles/`; los valores privados o puramente locales quedan fuera de Git.
