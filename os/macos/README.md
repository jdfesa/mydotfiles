# macOS

Configuraciones, scripts y administracion de paquetes que solo tienen sentido
en macOS.

Contenido inicial:

- `aerospace/`: window manager;
- `borders/`: JankyBorders;
- `hammerspoon/`: automatizaciones Lua;
- `sketchybar/`: barra de estado;
- `packages/homebrew/`: manifiestos de Homebrew.

Las herramientas compartidas con GNU/Linux o Windows pertenecen a `config/`, no
a esta carpeta.
