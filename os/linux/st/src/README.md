# ST personal build

Arbol fuente de ST mantenido por estos dotfiles. Incluye transparencia,
Xresources con recarga, ligatures, scrollback y `anysize`.

La configuracion actual vive en `config.h`. ST todavia debe recibir el mismo
flujo reproducible de build, prueba e instalacion que DWM y dmenu antes de
considerarse completamente migrado.

Atajos actuales relevantes:

- `Alt+c` y `Alt+v`: copiar y pegar;
- `Alt+a` y `Alt+s`: aumentar y reducir transparencia;
- `Alt+Shift+k` y `Alt+Shift+j`: aumentar y reducir la fuente.

No clonar ni instalar otra configuracion externa encima de este arbol.
